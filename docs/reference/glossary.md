# Glossary

- **Dataset**: A loaded spectrum (uploaded or fetched) plus metadata and provenance.
- **Trace**: A plotted series derived from a dataset (original or derived).
- **Derived trace**: A trace computed from one or more traces (e.g., Aâˆ’B, A/B).
- **Provenance**: Where data came from and what transformations were applied.
- **Citation-first**: Any scientific reference data used by the app is linked to a trusted source.
